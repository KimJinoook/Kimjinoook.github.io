## 1. 조건문   
```javascript
if(조건식1) {
  조건식1이 참인 경우 실행 내용;
}else if(조건식2) {
  2가 참인 경우 실행내용;
}else {
  조건을 모두 만족하지 않는 경우 실행 내용;
}
```   

## 2. 반복문
### while문   
```javascript
while(반복조건) {
  조건이 참인 동안 계속 반복할 내용;
}
```

### do while문
```javascript
 do {
  반복할 문장;
}while(조건);
```
- 일단 한번 실행한후 조건을 체크   

### for문
```javascript
for(초기화식; 조건식; 증감식) {
  반복될 문장;
}
```
