# 클래스
## 1. 객체 지향 프로그래밍
- 객체 (object)
  - 물건, 대상 등을 의미
  - 상태정보(속성, 데이터) / 변수
  - 행동(기능) / 메서드
- 객체 지향 프로그래밍
  - 현실에 존재하는 사물, 대상, 그에 따른 행동 등을 실체화 시키는 형태의 프로그래밍
    - 나는 상점에서 물건을 두개 구매했다
    - 나, 상점, 물건 / 객체
    - 사과 두개 / 상태
    - 구매 / 행동
```java
class apple{
  public static void main(String[] args){
    int apple = 20; // 사과의 수, 상태정보, 변수
    int money = 0; // 판매수익, 상태정보, 변수

- 객체를 생성하기전, 생성을 위한 틀을 만들어야 한다.
  - 객체 붕어빵 / 틀 붕어빵틀
  - 객체 변수, 메서드 등 / 틀 클래스
```
