## 특징
#### 감싸인 요소
- 컴포넌트에 여러 요소가 있다면, 반드시 부모요소 하나로 감싼 후 리턴해야 한다
  - 컴포넌트 내부는 하나의 DOM 트리 구조로 이루어져야 한다는 규칙
- v16 이후부터는 div태그 대신 Fragment 기능 사용 가능   

```javascript 
function App() {
  return (
    <h1>에러</h1>
    <h2>에러</h2>
  );
}
export default App;

function App() {
  return (
    <div>
      <h1>정상</h1>
      <h2>정상</h2>
    </div>
  );
}
export default App;


import { Fragment } from "react";
function App() {
  return (
    <Fragment>
      <h1>정상</h1>
      <h2>정상</h2>
    </Fragment>
  );
}
export default App;

function App() {
  return (
    <>
      <h1>정상</h1>
      <h2>정상</h2>
    <>
  );
}
export default App;

```
