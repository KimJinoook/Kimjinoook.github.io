# JSX
- 자바스크립트의 확장 문법
- 번들링되는 과정에서 자바스크립트 형태의 코드로 변환된다   
- jsx   

```javascript
function App(){
  return (
    <div>
      Hello <b>react</b>
    </div>
  );
}
```   

- 변환 후   

```javascript
function App(){
  return React.createElement("div",null,"Hello",React.createElement("b",null,"react"));
}
```
